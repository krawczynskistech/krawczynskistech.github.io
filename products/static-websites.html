Static websites

Static websites are the most basic type of website. The content of them is hard coded, which means that only the developer can change it directly inside the HTML file. There is not administative page in this type of website. Static websites are the best for landing pages, without any systems like chats, user login/registration etc. They don’t require a database and a backend service , which makes them cheaper.

A static website will require registering a github account for you which is completly free.
